# 🎉 儿童任务激励网站

一个专为 4 岁小朋友设计的任务激励平台，通过虚拟宠物养成系统提高孩子完成任务的积极性。

## 🌟 核心功能

### 儿童端
- 🏠 **首页**：展示今日任务和愿望单进度
- 📋 **任务打卡**：完成日常任务获得星星奖励
- 🐾 **宠物互动**：喂食、洗澡、玩耍，培养责任心
- 🛒 **商城兑换**：用星星购买宠物用品
- 🏅 **荣誉墙**：展示获得的徽章和奖杯
- 🎯 **愿望单**：设定目标并追踪进度

### 家长端
- 📊 **数据统计**：查看孩子任务完成情况
- 📝 **任务管理**：自定义任务内容和奖励
- 🛍️ **商城管理**：管理虚拟和实体商品
- 🐶 **宠物监控**：查看宠物状态
- 🎁 **愿望审核**：管理孩子的愿望清单

## 🚀 快速启动

### Mac / Linux 用户
```bash
chmod +x start.sh
./start.sh
```

### Windows 用户
双击 `start.bat` 或在命令行运行：
```cmd
start.bat
```

### 或者使用 Python 直接启动
```bash
python start.py
```

## 🎮 使用说明

### 家长设置流程
1. 登录家长端
2. 为孩子创建日常任务（如：早起、吃早餐、上学不迟到）
3. 设置每个任务的星星奖励
4. 在商城中添加可兑换的物品
5. 帮助孩子设定愿望单目标

### 儿童使用流程
1. 登录儿童端
2. 查看今日任务并完成打卡
3. 获得星星奖励
4. 照顾宠物（喂食、清洁、玩耍）
5. 用星星购买物品或实现愿望

## 🐾 宠物系统

宠物有三种状态值：
- 🍖 **饱腹度**：通过喂食提升
- ✨ **清洁度**：通过洗澡提升
- 😊 **心情值**：通过玩耍提升

状态值会随时间下降，需要孩子定期照顾宠物！

## 🏆 奖励体系

- ⭐ **星星**：日常任务奖励，用于消费
- 🏅 **徽章**：长期成就奖励
- 🏆 **奖杯**：特殊目标达成奖励

## 📝 任务类型

1. **日常任务**：每天自动刷新
2. **宠物任务**：宠物状态低时触发
3. **自定义任务**：一次性特殊任务
4. **强化任务**：复习机制，定期提醒
