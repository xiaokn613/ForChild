## 🎯 核心亮点

### 1. 游戏化激励机制
- 虚拟宠物养成 + 任务打卡结合
- 三种状态值需要孩子持续关注
- 宠物等级系统提供长期目标

### 2. 多层次奖励体系
- ⭐ 星星：日常激励
- 🏅 徽章：成就认可
- 🏆 奖杯：里程碑奖励

### 3. 情感连接设计
- 宠物会"饿"、会"脏"、会"不开心"
- 孩子通过照顾宠物培养责任心
- 宠物状态与孩子行为挂钩

### 4. 家长监管系统
- 完全的任务自定义权限
- 实时的数据统计
- 透明的消费记录
- 愿望单审核机制

### 5. 极简部署方案
- 一键启动
- 无需配置数据库
- 单文件数据存储
- 跨平台兼容

## 🚀 快速使用指南

### 第一步：启动应用
```bash
# Mac/Linux
./start.sh

# Windows
start.bat
```

### 第二步：访问系统
- 儿童端：http://localhost:5001/child/login
- 家长端：http://localhost:5001/parent/login

### 第三步：登录测试
**儿童端：**
- 用户名：child
- 密码：123456

**家长端：**
- 用户名：parent
- 密码：123456

### 第四步：开始使用

**家长操作流程：**
1. 登录家长端
2. 查看默认任务
3. 根据需要修改任务
4. 查看商城物品
5. 帮助孩子设定愿望

**儿童操作流程：**
1. 登录儿童端
2. 查看今日任务
3. 完成任务打卡
4. 获得星星奖励
5. 照顾宠物
6. 购买物品
7. 推进愿望进度
